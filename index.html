Funções

A funções facilitam na organização e otimização do código. Podemos criar funções para difersas finalizades, uma vez criadas podem ser usadas em diferentes partes do código facilitando na codificação.

Em python para definirmos uma função utilizamos a palavra def. E em seguida sempre acompanhado dos parênteses e dois pontos.

def(): # Definindo uma função

Toda função possui parâmetros, que são colocados dentro dos parentêses, responsáveis por receber valores quando a função é chamada, podemos passa os diversos tipo de dados para que a função processe estes dados e retorne algum valor.

def(valor1,valor2): # A função irá receber os valores valor1,valor2 quando chamada.

Agora que já foi definida a função e também já foi passado os parâmetros. Agora vamos montar nosso código de processamento. Como exemplo vamos criar o código de processamento onde a função irá executar a soma de dois valores valor1 e valor2 recebidos como parâmetro na função.

def somar(valor1,valor2): # processamento resultado = valor1 + valor2

Após montar o código de processamento, acionamos o Return que será responável por retornar algo, neste caso será retornado o resultado da soma.

def somar(valor1,valor2): # processamento resultado = valor1 + valor2 return resultado

Estrutura de dados

Existem estrutura de dados nos mais diversos tipos de linguagem como javascript,java,php e entre outras. O que muda é a maneira como são declaradas. Logo abaixo temos um exemplo que declara uma lista, uma tupla ou um dicionário de forma mais simples.

lista = [] # Declaração de uma lista

tupla = () # Delaração de uma tupla

dicionário = {} # Declaração de um dicionário

Podemos ainda declarar de uma forma mais detalhada deixando explicito a extrutura que está sendo definida.

lista = list() # Declaração de uma lista mais detalhada

tupla = tuple() # Delaração de uma tupla mais detalhada

dicionário = dict({}) # Declaração de um dicionário mais detalhado

Lista

Lista é um conjunto sequencial de elementos que podem ter valores de qualquer tipo, strings,float, int, incluindo outras listas.

lista = [10,10,2,3,9,4,8] # Lista

Exibindo a lista

Para exibir a lista toda utilizamos o comando print(lista)

print(lista) # Saida : [10,10,2,3,9,4,8]

Exibindo um elemento da lista

Cada elemento possui seu indices. Os indices iniciam na posição 0. Para exibir um dos elementos, basta chamarmos a função print() e passar como parâmetro a varialvel lista com o indice do elemento que deverá ser exibido print(lista[4])

lista = [10,10,2,3,9,4,8] # Lista
print(lista[4]) # Saida : 9

Tuplas

As tuplas são diferentes de listas, pois uma vez criadas, não aceitam modificações. Exitem duas formas de declarar uma tupla, uma mais detalhada que a outra.

tupla =() # Forma implicita tupla =tuple() # Forma Explicita

Agora podemos criar uma tupla com valores :

tupla =("Arroz","Feijão","Macarrão") # Forma implicita

Podemos acessar um elemento na tupla da mesma forma que fazemos na lista, apotando o seu indice.

tupla =("Arroz","Feijão","Macarrão") # Forma implicita print(tupla[0]) Retorna : Arroz

Podemos acessar ainda mais de um elemento na tupla informando o indice inicial e final.

print(tupla[0:2]) Retorna : Arroz,Feijão, Macarrão

Dicionário

Dicionários são estruturas de chave-valor, ou seja, os valores(dados) estão sempre associados a uma chave.

Declarar um dicionáro

Assim como lista e tuplas. os dicionários podem ser declarados de forma Implicita ou Explicita

dicionario = {} # Forma Implicita dicionario = dict({}) # Forma Explicita

Ao contrário das listas e tuplas, que seu indices não precisam ser declarados para criar uma lista ou tupla, apenas quando for acessar é necessário informa-lo. O Dicionário quando criado, é necessário associar seu indice ou chave para ser criado.

localizacao = { "Lat": -22.817087, "Long": -47.069750 }

Como podemos observar no códigos acima os valores "-22.817087,-47.069750" foram associados as chaves "Lat,Long" respectivamente.

Arquivos

Python dá a possibilidade de trabalharmos com arquivos de texto, sejam eles os mais variados possível, como arquivos de uma liguagem de programação como python ou html, como também arquivos de texto como de word ou de um bloco de notas, arquivos csv, tudo que se possa imaginar que são criados através de texto. O python consegue acessar estes arquivos e consegue executar operações como leitura e escrita.

A função open() é responsável por abrir um arquivo ou criar um se o arquivo não existe.

arquivo = open("nomedoarquivo.txt","r",encoding="utf-8") # Função responsável por abrir um arquivo de texto

Na função open() passamos três parâmetros :

  • 1º nome do arquivo que podem ser nos diferentes formatos como html,py,csv.txt,doc , entre outros.
  • 2º Permissão – Quais serão as permissões que este arquivos podem executar quando abertos, ou seja “w” para escrever, “r” para ler, ”r+” para ler e escrever, e “a” para anexar no final do documento.
    • w – Sempre vai criar um arquivo novo se o arquivo não existir, ou vai sobrescrever o arquivo caso já exija, ou seja toda informação anterior será perdida.
    • r – permite apenas a leitura do arquivo, se o arquivo não existir apresentará um erro.
    • r+ - permite a leitura do arquivo ou a escrita nele, não subscreve, mas todos os textos são adicionados no início. Se o arquivo não existir também cria um.
    • a – permite adicionar um texto no final do arquivo. Não subscreve.
  • 3º Encoding - Como sabemos que o conjunto de caracteres muda de acordo com a nacionalidade, utilizamos "utf-8” para nosso padrão.

O Nosso exemplo abaixo vamos abrir o arquivo texto.txt no modo leitura passando como parâmetro a letra r e também o tipo de caractere como UTF-8. Outros parâmetros podem ser passados conforme as permissão acima.

O arquivo texto.txt deve estar escrito algo e também deve estar salvo no mesmo local do seu arquivo python que está sendo executado.

Após aplicarmos a função open(), vamos chamar metodos responsáveis por ler ou escrever de acordo com a permissão que foi passada como parâmentro para a função open()

Para leitura de um arquivo utilizamos o metódo read().

arquivoTexto = open("texto.txt","r",encoding="utf-8") #Abertura do arquivo ler = arquivoTexto.read() # Acionammos a variável que armazenou a abertura do arquivo + o método read().

Para escrita de um arquivo utilizamos o metódo write() passando como parâmetro seu texto que deve ser escrito no arquivo.

arquivoTexto = open("texto.txt","r",encoding="utf-8") #Abertura do arquivo escrever = arquivoTexto.write("texto...") # Acionammos a variável que armazenou a abertura do arquivo + o método write().

Para exibir o que foi lido ou escrito em um arquivo, basta chamar a função print() passando como parâmetro a varivel que armazenou a escrita ou leitura do documento.

arquivoTexto = open("texto.txt","r",encoding="utf-8") #Abertura do arquivo escrever = arquivoTexto.write("texto...") # Acionammos a variável que armazenou a abertura do arquivo + o método write(). print(escrever) # Saida : texto do arquivo
