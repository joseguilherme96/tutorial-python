Estrutura de dados

Existem estrutura de dados nos mais diversos tipos de linguagem como javascript,java,php e entre outras. O que muda é a maneira como são declaradas. Logo abaixo temos um exemplo que declara uma lista, uma tupla ou um dicionário de forma mais simples.

lista = [] # Declaração de uma lista

tupla = () # Delaração de uma tupla

dicionário = {} # Declaração de um dicionário

Podemos ainda declarar de uma forma mais detalhada deixando explicito a extrutura que está sendo definida.

lista = list() # Declaração de uma lista mais detalhada

tupla = tuple() # Delaração de uma tupla mais detalhada

dicionário = dict({}) # Declaração de um dicionário mais detalhado

Lista

Lista é um conjunto sequencial de elementos que podem ter valores de qualquer tipo, strings,float, int, incluindo outras listas.

lista = [10,10,2,3,9,4,8] # Lista

Exibindo a lista

Para exibir a lista toda utilizamos o comando print(lista)

print(lista) # Saida : [10,10,2,3,9,4,8]

Exibindo um elemento da lista

Cada elemento possui seu indices. Os indices iniciam na posição 0. Para exibir um dos elementos, basta chamarmos a função print() e passar como parâmetro a varialvel lista com o indice do elemento que deverá ser exibido print(lista[4])

lista = [10,10,2,3,9,4,8] # Lista
print(lista[4]) # Saida : 9

Métodos para Manipulação de dados

Alguns métodos que auxiliam na adição, edição e remoção dos dos dado dentro de uma lista, tupla ou dicionário

Função Descrição Exemplo
len # Retorna o tamanho da lista l=[1,2,3,4]
len(l) # Saida : 4
min # Retorna o menor valor da list l=[10,40,30,20]
min(l) # Saida : 10
max # Retorna o maior valor da lista l=[10,40,30,20]
max(l) # Saida : 40
sum # Retorna a soma dos elementos da lista l=[10,40,30,20]
sum(l) # Saida : 100
append Adiciona um novo valor no final da lista l=[10,40,30,20] l=[10,40,30,20]
l.append(100) # Adiciona o valor 100 ao final da lista
print(l) # Saida : [10,40,30,20,100] com o valor já adicionado
extend # Insere uma lista no final de outra lista l=[10,40,30,20,100]
l.extend([5,6,15]) # Adiciona lista no final da outra lista
print(l) # Saida : [10,40,30,20,100,5,6,15] já com a outra lista já adicionado

del l[0] # Deleta o elemento com o indice
print(l) # Saida : [40,30,20,100,5,6,15] sem o elemento 10 que estava no indice 0
in # Verifica se o valor pertence a lista l=[10,40,30,20]
10 in l # Saida : True, pois 10 está na lista
sort # Ordena os calores da lista em ordem crescente l=[10,40,30,20] l.sort() lista=[10,40,30,20]
lista.sort() # Ordena em ordem crescente
print(lista) # Saida :[10,20,30,40] já a lista em ordem crescente.
